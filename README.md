# CareLink

CareLink is a mobile app designed to support caregivers of patients with dementia. It helps family members, friends, and professional caretakers manage care efficiently and stay connected.

---

## Features

- User accounts for caregivers
- Chronological feed of entries to track patient information over time
- Simple entry form to add new updates efficiently
- Calendar view for a quick overview of past entries
- Report tab with an AI-powered tool to generate summaries for any timeframe


---

## Tech Stack

- **Frontend:** React Native / Expo  
- **Backend:** Supabase

---

## Getting Started

Follow these steps to run CareLink locally:

1. Install dependencies:

  ```bash
  npm install
  ```

2. Start the development server:

  ```bash
  npx expo start
  ```
