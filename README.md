# CareLink

CareLink is a mobile app designed to support caregivers of patients with dementia. It helps family members, friends, and professional caretakers manage care efficiently and stay connected.

---

## Features

- Track patient care and daily routines
- Communicate with other caregivers
- Manage tasks, appointments, and reminders
- Support multiple caregiver roles (family, friends, professionals)


---

## Tech Stack

- **Frontend:** React Native / Expo  
- **Backend:** Supabase

---

## Getting Started

Follow these steps to run CareLink locally:

1. Install dependencies:

  ```bash
  npm install
  ```

2. Start the development server:

  ```bash
  npx expo start
  ```
